# SpaceX_Capstone
#### This capstone project uses a number of machine learning classification techniques to forecast if the SpaceX Falcon 9 first stage will land successfully. This project's primary phases are data gathering, processing, and formatting; exploratory data analysis, interactive data visualization as well as machine learning prediction.
#### The graphs demonstrate a relationship between certain aspects of rocket launches and their results.
